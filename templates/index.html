How Metrics Are Computed

  • Vehicle Count: YOLO11s detections for vehicles plus emergency ambulance detector
  • Queue: Vehicles counted when density > 60%
  • Density: count / max_capacity (20 vehicles)
  • Wait Time: density × (1 - speed_factor) × 120s
  • Throughput: vehicle count x speed factor x 3 vehicles/min proxy

RL Decision Pipeline

  • Observation: 17-dim vector (4x queue, density, speed + phase + time)
  • Model: PPO with VecNormalize (or max-pressure fallback)
  • Actions: 0/1 = NS straight/left, 2/3 = EW straight/left
  • Live Guard: Max-pressure and emergency vehicle priority override unsafe PPO choices
  • Emergency Guard: ambulance detections preempt to the matching straight green and hold briefly
  • Static baseline uses 30s minimum green and 60s maximum green for comparison.
